# Number System Converter (Decimal to Binary, Octal & Hexadecimal)

This is a simple **Python console program** that accepts a **decimal number** from the user and converts it into:

- Binary
- Octal
- Hexadecimal

The user can choose the desired number system, and the result is displayed on the terminal.

---

##  Features
- Takes decimal input from the user
- Menu-driven program
- Uses Python inbuilt functions:
  - `bin()` for Binary
  - `oct()` for Octal
  - `hex()` for Hexadecimal
- Beginner-friendly and easy to understand

---

## How It Works
1. User enters a **decimal number**
2. User selects:
   - `1` → Binary
   - `2` → Octal
   - `3` → Hexadecimal
3. Program converts and displays the result
4. Shows an error message for invalid choices

---
